Detailed Description

template<class T, uint16_t nSize>
class BufferTemplate< T, nSize >

Classe template d'un Buffer de type fifo dont le type et le size sont configurable T est le type des données du buffer nSize est le nombre d'éléments du buffer. nSize doit absolument être une puissance de 2 (ex : 16,32,64)
